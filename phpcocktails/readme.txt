English -----------------------------------------------------------------------
                  Installation instructions for phpcocktails

1. extract files from the downloaded file (using tar or unzip command, or
   softwares like powerarchiver or winzip)

2. place de source files on your website in the directory of your choice
   ("phpcocktails" for example), it is useless to place "readme.txt" file on your
   website

3. go to the URL http://your.domain/phpcocktails/admin/install.php

4. follow the instructions of installation

5. delete admin/install.php and readme.txt files

Francais ----------------------------------------------------------------------
                 Instructions d'installation de phpcocktails

1. décompresser à l'aide de winzip par exemple (winrar, winace et beaucoup
   d'autres le permettent également) le fichier téléchargé.

2. placer les fichiers décompressés sur votre serveur web dans le répertoire
   de votre choix ("phpcocktails" par exemple), il est inutile de placer le
   fichier "readme.txt" sur votre serveur web

3. aller à l'URL http://votre.domaine/phpcocktails/admin/install.php

4. suivre les instructions d'installation

5. supprimer le fichiers admin/install.php et readme.txt
